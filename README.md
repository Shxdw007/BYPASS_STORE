# BYPASS_STORE
https://shxdw007.github.io/BYPASS_STORE/


### 📝 Описание проекта
**BYPASS_STORE** — это веб-интерфейс для интернет-магазина, разработанный с использованием современных веб-технологий. Проект включает в себя каталог товаров, систему навигации и функционал корзины.

---

### 🛠 Технологический стек
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
* **Media:** Интеграция видео-контента (`trailer.mp4`).
* **Version Control:** Git & GitHub.

---

### 📂 Структура проекта
* `index.html` — Главная страница магазина.
* `shop.html` — Каталог доступных товаров.
* `cart.html` — Страница оформления заказа и корзина.
* `style.css` — Основные стили и адаптивная верстка.
* `script.js` — Логика взаимодействия пользователя с сайтом.
